# Absensi Karsa - VS Code / Vite React

Versi baru aplikasi absensi karyawan Karsa dengan header home putih, tanpa box hijau.

## Jalankan di Visual Studio Code

```bash
npm install
npm run dev
```

Buka:

```text
http://localhost:5173
```

## Catatan

- Kamera dan GPS paling aman dijalankan di `localhost` atau domain HTTPS.
- Endpoint Google Apps Script sudah dipasang di `src/App.jsx`.
- Header home terbaru tidak lagi memakai kartu gradient hijau.
